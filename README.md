## 🎯 What This Badass Little Backend Does

This isn't just code – it's **proof of concept** that I can dive headfirst into technical challenges and emerge victorious. Here's what I built in 24 hours:

### 🏗️ **Tech Stack** (Because I'm fancy now)
- **FastAPI**: Lightning-fast Python web framework
- **SQLite**: Because sometimes simple is genius
- **Pydantic**: Data validation that doesn't mess around
- **Faker**: For generating realistic test data like a boss

### 🔥 **API Endpoints**
- `GET /` - A friendly "Hello Leslie" (because personal branding matters)
- `GET /items/{item_id}` - Dynamic item retrieval with optional query params
- `GET /customers` - Full customer data dump from the database

### 📊 **Database Design**
Built a complete business database schema with:
- **Customers** (100 fake records, all validated)
- **Products** (50 items across 7 categories)
- **Employees** (25 team members with realistic data)
- **Orders & Order Items** (200 transactions with proper relationships)

### 🧠 **Smart Features**
- **Data aggregation** with SQL GROUP BY operations (`groupby.py`)
- **Custom customer insertion** with proper validation (`queries.py`)
- **Automated database population** with realistic business data
- **Built-in reporting** for sales analytics and employee performance

## 🎪 The Files (And What They Actually Do)

- **`main.py`**: The FastAPI application that makes everything tick
- **`database.py`**: The beast that creates, populates, and reports on our entire business database
- **`queries.py`**: Custom database operations (featuring a cheeky "Hallo Voids" customer entry 😉)
- **`groupby.py`**: SQL aggregation magic for data analysis
